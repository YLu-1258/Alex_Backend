What is a sorting algorithm?

  1. A sorting algorithm is a set of logic and rules created to sort a collection of similar and related elements.
  2. It commonly has uses in ordering collections of data to make certain elements more accessible, or to create certain properties that allow for certain operations, like taking every other element in ascending order.
  3. It covers commonly used algorithmic techniques like recursion, array pointers, and divide and conquer.
